# Stardew Valley Prototype

当前仓库处于原型前期部署阶段。

## 当前目标

先完成一个可玩的农场垂直切片，验证基础农场循环，而不是直接扩展完整内容。

## 目录说明

- `docs/`：项目约定、原型范围、系统分工说明
- `scenes/`：场景文件
- `scripts/`：GDScript 代码
- `resources/`：数据资源目录
- `assets/`：美术与音频资源目录
- `tests/`：测试目录预留

## 开发入口

并行开发前，先阅读：

1. `docs/project_shared_conventions.md`
2. `docs/prototype_scope.md`
3. 对应系统的 `docs/system_briefs/*.md`
